6. Are there differences in activity patterns between weekdays and weekends?

For this part the weekdays() function may be of some help here. Use the dataset with the filled-in missing values for this part. Create a new factor variable in the dataset with two levels – “weekday” and “weekend” Make a panel plot containing a time series plot (i.e. type = “l”) of the 5-minute interval (x-axis) and the average number of steps taken, averaged across all weekday days or weekend days (y-axis). See the README file in the GitHub repository to see an example of what this plot should look like using simulated data.

filledData$Day <- weekdays(filledData$date)
for (i in 1:nrow(filledData)){
        if (filledData$Day[i] == "Saturday" | filledData$Day[i] == "Sunday"){
                filledData$Day[i] = "weekend"
        }
        else{
                filledData$Day[i] = "weekday" 
        }
}
filledData$Day <- as.factor(filledData$Day)
stepsPerIntervalFilled <- aggregate(steps ~ interval + Day, data = filledData, mean)
library(lattice)
xyplot(steps ~ interval | Day, stepsPerIntervalFilled, type = "l", layout = c(1, 2), xlab = "Interval", ylab = "Steps")
